# МебельЭко - Сайт мебельной мастерской

Этот проект представляет собой веб-сайт для мебельной мастерской "МебельЭко", специализирующейся на изготовлении мебели на заказ из натуральных материалов.

## Демо

Сайт доступен по адресу: [https://dmitriy-bogomolov-fly.netlify.app/](https://dmitriy-bogomolov-fly.netlify.app/)

## Технологии

Проект разработан с использованием следующих технологий:

- **React** - библиотека для создания пользовательских интерфейсов
- **TypeScript** - типизированный JavaScript для более надежного кода
- **Tailwind CSS** - утилитарный CSS-фреймворк
- **Vite** - быстрый сборщик и сервер разработки
- **Bun** - JavaScript-среда выполнения и менеджер пакетов

## Основные разделы сайта

1. **Главная** - с баннером и кратким описанием услуг
2. **Портфолио** - галерея работ с фильтрацией по категориям
3. **Услуги** - описание предоставляемых услуг
4. **О нас** - информация о компании
5. **Контакты** - контактные данные и форма обратной связи

## Запуск проекта локально

1. Клонировать репозиторий:
```bash
git clone https://github.com/Lyti4/dmitriy-bogomolov-fly.git
cd dmitriy-bogomolov-fly
```

2. Установить зависимости:
```bash
bun install
```

3. Запустить сервер разработки:
```bash
bun run dev
```

4. Открыть [http://localhost:5173](http://localhost:5173) в браузере

## Сборка для production

```bash
bun run build
```

## Деплой

Проект настроен для автоматического деплоя на Netlify при пуше в ветку main.
